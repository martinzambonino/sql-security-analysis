# Security Policy

## Educational Scope

The tools and code contained in this repository are developed strictly for academic, research, and educational purposes. They are intended for use only in simulated, contained environments or against systems where explicit consent has been granted.

## Reporting a Vulnerability

If you discover a security vulnerability within this project itself, please do not disclose it publicly.

Instead, open an issue labeled `security` or contact the repository owner directly. Given the educational nature of the project, reasonable effort will be made to remediate architectural or dependency vulnerabilities.

## Responsible Use

By cloning or downloading materials from this repository, you agree to:

1. Not use the materials for malicious or unauthorized activities.
2. Only execute queries against databases you own or have explicit written permission to access.
3. Assume full responsibility for compliance with all applicable local and international laws.
4. Acknowledge that these queries and techniques are intentionally simplified for learning and are not production-grade.
